## JavaScript 문법

일반적으로 js를 참조하게 되면 한줄씩 parsing하여 DOM 요소로 변환함.

스트립트 태그를 발견하면, 서버에서 해당 파일을 다운로드 받게 됨(latency가 발생).

### 기본 참조방식

```javascript
<head> or <body>
    <script src="test.js"></script>
</head> or </body>
```
